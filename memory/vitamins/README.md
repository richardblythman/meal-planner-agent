# Vitamins

This directory contains information about vitamins relevant to meal planning and nutritional optimization.

## Purpose

- Document vitamin requirements and recommended daily intakes
- Track vitamin content in foods and recipes
- Support nutritional gap analysis in meal planning
- Inform elimination diet planning when investigating deficiencies

## Structure

Individual vitamin files can be added here with details on:
- Recommended daily intake
- Food sources
- Deficiency symptoms
- Interactions with other nutrients
- Notes on absorption and bioavailability

## Usage

Reference these files when:
- Analyzing meal plans for micronutrient completeness
- Identifying foods to address specific vitamin gaps
- Planning balanced weekly meals
